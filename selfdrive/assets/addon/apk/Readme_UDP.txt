0. The apps can parse the navigation data using UDP broadcast.
1. Install the modified Tmap or Waze on your smartphone. (No rooting required)
2. In the C3 or C3X UI menu, go to the navigation selection menu and choose Tmap-UDP or Waze-UDP.
3. The device and smartphone must be on the same network. (Supports private IP ranges: 192.x, 10.x, 172.x)
4. Now, when you launch the navigation app, it will automatically sync.

0. 네비 데이터를 UDP브로드캐스트를 이용하여 전송하는 방식입니다.
1. 수정된 티맵 혹은 웨이즈를 스마트폰에 설치합니다.(루팅 필요 없음)
2. C3 혹은 C3X UI메뉴 네비선택 메뉴에서 티맵-UDP 혹은 웨이즈-UDP를 선택합니다.
3. 장치와 스마트폰은 같은 네트워크에 있어야 합니다.(사설IP 192대역, 10대역, 172대역 지원)
4. 이제 네비 앱을 실행하면 자동으로 연동됩니다.
